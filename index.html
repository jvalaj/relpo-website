Relpo

An app that helps users to find and host acting gigs near them.

Download on the PlayStore

Services

Relpo includes many features that help you utilise the app to the fullest.

Inbuilt Chatting

Our inbuilt chatting feature allows all applicants to communicate with the hosts easily.

Host Gigs

You can host as many gigs as you want with all the details required such as the number of positions available, information about each role and the story.

Find Gigs

With relpo you can easily find gigs near you and cast for the perfect role.

User Authentication

We make sure that every user registers on the app and provides certain important information about them to make Relpo a safe platform.
